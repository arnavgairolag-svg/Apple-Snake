# Apple-Snake
Apple Snake is a Python arcade game using Pygame where players collect apples, avoid poison items, and survive enemy snakes. The game features leveling, lives, particle effects, and difficulty modes that increase the challenge as you progress.
✨ Features

🐍 Smooth snake movement
🍎 Apple collection system
☠️ Poison apples that reduce lives
❤️ Heart power-ups for extra lives
👾 Enemy snakes that move around the map
📈 Level progression with increasing difficulty
🎮 Easy, Normal, and Hard modes
✨ Particle explosion effects
🎨 Animated snake body wave movement
🖥️ Clean game over screen with final score

🖼️ Gameplay
Objective

Eat red apples to gain points and grow.

Avoid poison apples and enemy snakes.

Collect hearts to gain extra lives.

Survive as long as possible and reach higher levels.

🎮 Controls
Key	Action
⬆️ Up Arrow	Move Up
⬇️ Down Arrow	Move Down
⬅️ Left Arrow	Move Left
➡️ Right Arrow	Move Right
Space	Start game
⚙️ Requirements

Python 3

Pygame

Install pygame:

pip install pygame

🧠 Game Mechanics

Apples increase score and snake length

Poison apples remove lives

Hearts add extra lives

Levels increase the number of items and snake speed

Enemy snakes appear in higher levels and can steal apples

📊 Difficulty Modes
Mode	Speed	Poison
Easy	Slow	Few
Normal	Medium	Moderate
Hard	Fast	Many
